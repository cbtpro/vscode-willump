# Willump - VS Code 插件

Willump 是一款强大但不张扬的 VS Code 插件，专注于开发者的本地端口管理。

## ✨ 功能亮点

- ✅ 查看指定端口是否被占用
- 🛠️ 杀掉指定端口的进程
- 📊 查看当前系统所有被占用的端口信息

## 📦 命令列表

| 命令                            | 描述                         |
| ------------------------------- | ---------------------------- |
| `Willump: 检查端口占用情况`     | 输入端口号，查看是否被占用   |
| `Willump: 杀掉占用端口的进程`   | 输入端口号，终止该端口的进程 |
| `Willump: 查看所有端口占用情况` | 一键查看当前被占用的所有端口 |
